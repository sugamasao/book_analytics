
## How To Run

### Set ENV

- [access to amazon associate site](https://affiliate.amazon.co.jp)

```sh
export AMAZON_ACCESS_KEY=your key
export AMAZON_SECRET_KEY=your secret
export AMAZON_ASSOCIATE_TAG=your associate name
```
